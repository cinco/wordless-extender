    Wordless is not anymore at its first release. We belive that it has to be an always evolving piece of code, following technologies, devops needs, modern coding languages and conventions. Wordless is an opinionated Wordpress plugin that dramatically speeds up your custom theme creation.

We're trying to get Wordless fully documented, overall in his helpers collection.

If you only need documentation, for creating Wordless themes, open the Wordless documentation and surf to Modules -> Helpers Functions and Modules -> Vendor; you'll find all the helpers functions organized in easy to understand classes.
